# Live Financial News Sentiment Classifier & Analytics Pipeline

*   **[📁 View Complete Source Code Repository](https://github.com/Egla23/NLP--Financial-Sentiment-Predictor-)**

### Tech Stack and Libraries 
*   **Language:** Python
*   **AI/LLM Execution:** langchain-google-genai (utilizing gemini-2.5-flash)
*   **Data Parsing:** feedparser (RSS/XML processing)
*   **Data Validation:** pydantic (strict type enforcement and structured schema parsing)
*   **Data Visualization:** matplotlib (advanced grid layouts with gridspec)

###  Core Architecture
1.  **Live Data Ingestion (Live_agent.py):** Connects to the live Yahoo Finance RSS feed, filters out previously analyzed articles to prevent duplicates `seen_articles.json`, and extracts headlines and source URLs.
2.  **Structured NLP Classification (Live_agent.py):** Uses gemini-2.5-flash with Pydantic structured outputs to extract market sentiment, volatility risk, target tickers, a one-sentence trading thesis, and a classification rationale. It is specifically prompted to detect financial slang and sarcasm.
3.  **Performance Analytics (analysis.py):** Reads the processed history database (seen_articles.json) and generates a custom 4-panel data visualization report detailing sentiment trends, volatility exposure, top-mentioned tickers, and data completeness metrics.
